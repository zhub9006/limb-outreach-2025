## Quick Start
1. **Claim a task** → Browse the `tasks/` directory or search GitHub Issues
2. **Report findings** → Add to `clinical-trials/trials-index.md`
3. **Sync with team** → Check `meetings/coordinator-sync.md`
4. **Share resources** → Drop files in `outreach/`
